El script (Obtener-y-actualizar-estado-de-Firewall.ps1) permite a los usuarios obtener el estado de los perfiles del firewall (Dominio, Privado y Publico) ademas permite actualizar el estado de cada perfil (Habilitar o deshabilitar). Caracteristicas del Script:
- Obtener el estado de perfiles de firewall: La primera opción del script es obtener todos perfiles, proporcionando una visión del estado de cada uno de ellos.
- Interacción con el usuario: El script permite decidir entre la opción de obtener el estado y actualizar el estado de cada perfil del firewall.
- Actualizar el estado de perfiles de firewall: El script ofrece la opcion de habilitar o deshabilitar el estado de cada perfil, lo que permite al usuario gestionar el firewall de manera más efectiva y adaptarse a cambios en las politicas de seguridad.

+ El script se debe ejecutar en modo administrador.

The script (Obtener-y-actualizar-estado-de-Firewall.ps1) allows users to obtain the status of firewall profiles (Domain, Private, and Public) and update the status of each profile (Enable or Disable). Script features:
- Get firewall profile status: The script's first option is to retrieve all profiles, providing an overview of the status of each one.

- User interaction: The script allows users to choose between getting the status and updating the status of each firewall profile.

- Update firewall profile status: The script offers the option to enable or disable the status of each profile, allowing users to manage the firewall more effectively and adapt to changes in security policies.

+ The script must be run in administrator mode.
